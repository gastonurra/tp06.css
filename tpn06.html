¿A qué se llama regla de CSS?

Se llama patrón de elementos y otros términos que indican al navegador qué elementos HTML se seleccionan para aplicarles una regla que incluye los valores de las propiedades CSS.

¿Qué es un selector en CSS?

Selector: selecciona el(los) elemento(s) a dar estilo (en este caso, los elementos "p"). Para dar estilo a un elemento diferente, solo cambia el selector.

Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

Selector universal Nos permite aplicar estilos a todos los elementos HTML de un documento y se indica mediante el *. * { color:red; } Todos los elementos HTML tendrán un color de texto red. Selector por etiquetas Aplicamos propiedades CSS a una etiqueta. La misma se identifica sin utilizar >. Por ejemplo. p { color:red; } Podemos, seleccionar varias etiquetas al mismo tiempo concatenándolas con el signo , (coma). Por ejemplo: p,h1 { color:red; } En el ejemplo anterior, tanto la p como la h1 tendrán de color de letra roja. Reordar que se aplicarán a todas las etiquetas del documento HTML. Selector descendente Con este selector seleccionamos un elemento que está dentro de otro del mismo tipo. Estos seectores siempre están formados por dos o más selectores separados entre si. El último selector será el elemento HTML al que se le aplicará las propiedades de CSS. Por ejemplo: p span { color:green; } Selector de clases Los selectores de clases comienzan siempre por un . (punto). Los clientes web (navegadores) identifican este punto y lo distinguen del resto de los selectores. Por ejemplo:

Texto de un párrafo

Para definir en HTML el uso de los selectores de clase, se utiliza el atributo class. En css se aplica así: .primer-parrafo { font-size: 2em; } A todas las etiquetas que tengan la clase primer-parrafo, se aplicará un font-size (tamaño de letra) específico. Las clases pueden ser utilizadas varias veces y en distintas etiquetas, lo que nos hace pensar que es necesario ver cuales son la características que queremos aplicar a las etiquetas que sean comunes a varias. A su vez, podemos seleccionar un conjunto de etiquetas que selecione todas las etiquetas que tengan una clase particular. p.primer-parrafo { color: #445544; } En el ejemplo anterior, seleccionamos todos los párrafos que tengan la clase primer-parrafo y le aplicamos un color de letra especial. Selector de ID Este tipo de selector es igual al de clase, aunque lo aplicaremos a un único elemento HTML de nuestra página. Esto se debe a que el atributo ID de HTML identifica a un elemento HTML por medio de un nombre. Por ejemplo:

Para aplicar CSS solamente a ID titulo, tendremos que seleccionar dicho ID utilizar el # seguido del nombre: #titulo { color: violet; } De la misma forma que el caso anterior, podemos seleccionar un elemento basado en el id. Por ejemplo: a#titulo { color: red; } En el ejemplo anterior se aplicará la propiedad color a la etiqueta a (anchor) que tenga el ID #titulo.

"line-height" Define la altura de línea del texto. valores posibles normal, 1.5, 2, 20px, 1.2em
"color" Define el color del texto.valores posibles son los colores ejemplo red,green,white y hexadecimal
"background-color" Define el color del fondo del texto.valores posibles son los colores ejemplo red,green,white,yellow,hexadecimal y rgb
"font-size" Define el tamaño de la fuente del texto. valores posibles unidades absolutas,12px.unidades relativas 1em.palabras claves small medium
"font-weight" Define el grosor de la fuente del texto (por ejemplo, si se trata de texto en negrita o normal). valores posibles
"font-style" Define el estilo de la fuente del texto (por ejemplo, si es cursiva o normal). valores posibles normal italic oblique
"font-family" Define el tipo de letra a utilizar para el texto. valores posibles Tipos de fuente: "Arial", "Times New Roman", "Courier New" Grupos de fuentes genéricos: serif, sans-serif, monospace, cursive, fantasy
"text-align" Define la alineación horizontal del texto (izquierda, centro o derecha). valores posibles right,left,center y justify
"text-decoration" Define la decoración del texto (por ejemplo, si está subrayado o tachado). valores posibles none,underline,overline,line-through,blink
"text-transform" Define la transformación del texto (por ejemplo, en mayúsculas o minúsculas). valores posibles none,uppercase,lowercase,capitalize
"letter-spacing" Define el espacio entre letras. valores posibles normal, 2px, 0.1em, -0.05em
